Job Seekers Website for CCED Universitas Lampung

Project Overview

This job seekers website is a web-based application developed for the Center for Career and Entrepreneurship Development (CCED) at Universitas Lampung. The platform aims to facilitate efficient job search and application processes for students and alumni.

The system features user registration with role selection, secure login, user and company dashboards, and real-time updates on job seekers. The project was implemented using modern web technologies to ensure responsiveness, security, and a user-friendly experience.

This project was part of an undergraduate thesis focused on designing and building a reliable and easy-to-use job seekers information system.

Project Information

  • Category: Web Development
  • Client: Universitas Lampung
  • Date: 2024
  • Technologies: Next.js, Prisma ORM, NeonDB, Tailwind CSS
